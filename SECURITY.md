# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅ Yes    |

## Reporting a Vulnerability

**Do not report security vulnerabilities through public GitHub issues.**

To report a vulnerability:

1. Email the project maintainers at `miruchigawa@outlook.jp`
2. Include a description of the vulnerability.
3. Provide steps to reproduce the issue.
4. Include any potential impact details.

You will receive a response within 5 business days. If confirmed, we will release a patch and credit you in the release notes.
